Roteiro de prática


Para resolver um estudo de caso de modelagem a partir de modelo descritivo, no minimundo do enunciado você deve:

 

Identificar no modelo descritivo as entidades tipo e representá-las no DER.
 
Identificar no modelo descritivo os relacionamentos tipo e representá-los no DER.
 
Estabelecer as cardinalidades dos relacionamentos tipo e representá-las no DER.
 
Modelar os atributos.
 

Observe o DER de solução.
